### How to run

##### You will need to clone the project

```
git clone https://github.com/estaeldev/task-todo.git
```

##### Install dependencies and run the init script

```
yarn

yarn start
```

##### Start mock [**`Json Server`**](https://www.npmjs.com/package/json-server) backend

```
yarn mock_server
```
